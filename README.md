Personal Portfolio Website — Capstone Major Project

This is a responsive and interactive personal portfolio website created as a Capstone Project.  
It showcases my skills, projects, education, certifications, achievements, and contact details in a modern UI/UX format.

The website includes features like dark/light theme toggle, smooth scroll animations, loader animation making it visually engaging and professional for academic evaluation and real-world job applications.
